# DB-Hottel #

### Приложение базы данных отеля c СУБД PostgreSQL на WindowsForm. ###
------------------------------------------------------
###### Возможности: ######
* Сохраниение удаление пользователей приложения.
* Регистранция
* Авторизация
* Бронирование мест отеля 
* Чат с поддержкой
------------------------------------------------------
###### В проекте используются ######
* С#
* PostgreSQL
